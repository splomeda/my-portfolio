# my-portfolio

My University Portfolio

This is a multi-page portfolio website with pages that include the home/index page, an about page, a resume page, a projects page with corresponding pages for individual projects including a page for Sierra Crest and a page for Hype Movies, and a contact page.

This website uses HTML and CSS and has no javascript.
The HTML is semantic and validated.
